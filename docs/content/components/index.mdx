# Components

facebook.js provides a set of components to make integrating with Facebook easier. Below are some examples:

## Login Button

The Login Button component provides a simple way to add Facebook login to your application.

<Code language="js">{`
import { LoginButton } from 'facebook.js';

function MyApp() {
  const handleLogin = (response) => {
    console.log('User logged in:', response);
  };

  return (
    <LoginButton 
      appId="your-app-id"
      scope={['email', 'public_profile']}
      onSuccess={handleLogin}
      onError={(error) => console.error(error)}
    >
      Login with Facebook
    </LoginButton>
  );
}
`}</Code>

## Share Button

The Share Button allows users to share content on Facebook.

<Code language="js">{`
import { ShareButton } from 'facebook.js';

function ShareExample() {
  return (
    <ShareButton 
      url="https://example.com"
      quote="Check out this awesome website!"
      hashtag="#facebook"
    >
      Share on Facebook
    </ShareButton>
  );
}
`}</Code>

## API Methods

Beyond components, facebook.js also provides direct API methods:

<Command>
facebook.login(scopes)
facebook.logout()
facebook.getProfile()
facebook.share(options)
</Command>

<Tip>
For the best user experience, make sure to handle errors gracefully in your application when using these components.
</Tip>
