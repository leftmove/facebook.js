## Upload

<Documentation file="client/upload.ts" abstraction="Upload" showSearch={true} />

### Examples

```typescript
import Facebook from "bookface";

const facebook = new Facebook({ profile: "page" });

// Upload a single image
await facebook.upload.image({ media: "/path/to/image.jpg" });

// Upload multiple images
await facebook.upload.image({ media: [
  "/path/to/1.jpg",
  "/path/to/2.png"
]});

// Upload from Base64 data
await facebook.upload.image({ media: { data: "...base64...", type: "png" } });
```

## Post and Comment Objects

<Documentation file="client/posts.ts" abstraction="Post" showSearch={true} />

<Documentation file="client/comments.ts" abstraction="Comment" showSearch={true} />

### Examples

```typescript
// Edit and delete directly from a Post instance
const post = await facebook.page.posts.publish({ message: "Hello" });
await post.edit({ message: "Edited" });
await post.remove({});

// Reply to a comment from the Comment instance
const comment = await facebook.page.comments.publish({
  post: post.id,
  user: post.user!,
  message: "Nice post!"
});
await comment.reply({ id: comment.id, post: post.id, message: "Thanks!" });
```
